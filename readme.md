# Boilerplate-webpack
#### By [marioroca.dev](https://marioroca.dev)

This's a simple html, sass and js app that can be deployed with Vercel and zero configuration.

## How to run in Local

To start the app in local, first go to the following route:
and install node dependencies.

You have to have open terminals.

Recommended, use the latest stable version of node installed on your pc.

```shell
$ yarn install
```

then you need start de application:

```shell
$ yarn start
```


Finally, the terminal will show the path to test changes locally.

_localhost: 3000_
